# 🧮 Simple Web Calculator

A sleek, responsive, and functional calculator built with vanilla web technologies. This project demonstrates clean UI/UX design principles and logical state management using JavaScript.

## 🚀 Live Demo
Check out the live application here: 
[**View Live Project**](https://krishNA2000000.github.io/Simple-Calculator/)

## ✨ Features
* **Real-time Calculation:** Instant processing of basic arithmetic (Addition, Subtraction, Multiplication, Division).
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Error Handling:** Prevents invalid mathematical operations (like dividing by zero).
* **Clean UI:** Minimalist interface built with modern CSS.

## 🛠️ Tech Stack
* **HTML5:** Structured the semantic layout.
* **CSS3:** Custom styling with Flexbox/Grid for responsiveness.
* **JavaScript (ES6):** Implemented the calculation logic and DOM manipulation.
